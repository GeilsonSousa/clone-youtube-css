Seletores CSS

Bem Vindo!

No CSS nos possuimos seletores com ID

  • Seletor por tipo:Busca Elementor por uma Tag HTML específica
  • Seletor por ID(#):Busca elementos atraves do atributo "id"
  • Seletor por Classe:(.)Busca elementos atraves do atributo "class"
  • Seletor Universal(*):Seleciona todos os elementos HTML
  • Seletor de Atributo([atrib]):Seleciona elementos que possuem um atributo especifico em nosso HTML e conseguimos também buscar atributos com um valor especifico
